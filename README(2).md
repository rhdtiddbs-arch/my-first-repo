# 🎮 Linux Command Quiz Game (Python)
파이썬을 활용하여 첫 번째 과제에서 학습한 리눅스 핵심 프로세스 명령어(`top`, `ps`, `jobs`, `kill`)를 재미있게 복습할 수 있는 퀴즈 프로그램입니다.
---
## ✨ 주요 기능 및 특징 (본인 구현 논리)
단순한 일방향성 퀴즈 코드를 탈피하여, 사용자와 상호작용할 수 있는 제어 흐름 구조를 직접 설계했습니다.
1. **랜덤 출제 시스템 (`random.shuffle`)**
   * 게임을 실행할 때마다 문제의 순서가 무작위로 변경되어 반복 학습이 가능합니다.
2. **목숨(Life) 제한 시스템 (`while`, `break`)**
   * 사용자에게 총 3번의 기회가 주어지며, 정답을 틀려 목숨이 0이 되면 게임이 즉시 종료되는 조건문 루프를 구현했습니다.
3. **실시간 힌트 기능 (`Hint`)**
   * 문제 유형을 모를 때 `hint`를 입력하면 해당 명령어의 글자 수 문자열 길이를 계산하여 힌트를 동적으로 제공합니다.
4. **동적 피드백 시스템**
   * 최종 퀴즈 점수에 따라 사용자의 등급(`마스터`, `주니어`, `초보자`)을 다르게 출력하는 분기 처리를 적용했습니다.
